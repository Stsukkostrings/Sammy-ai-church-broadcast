Welcome

About The Product

A church broadcast workspace built for notes, scripture, planning, and presentation prep.

This web app mirrors the Android version while staying lightweight enough for volunteers, media desks, and pastors using a laptop or browser.

What It Does

  • Continuous mic listening in supported browsers
  • Editable sermon notes with download and share
  • Bible verse lookup for overlays
  • Session archive for sermon history
  • Service planner for media execution
  • Presentation preparation tools

Best Use Case

  • Small to mid-sized church media teams
  • Pastor sermon capture and note prep
  • Volunteer broadcast coordination
  • Scripture and content prep before OBS integration
